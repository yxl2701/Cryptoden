# Cryptoden

项目地址：https://gitee.com/yxl2701/cryptoden

Cryptoden 是一个面向 CTF 密码学题目的加解密工具，支持图形界面、命令行、一键解密、递归解密和常见攻击辅助。

## 特点与优势

- 上手快：提供图形界面，不熟悉命令行也能直接使用。
- 适合比赛：覆盖 CTF 密码题中常见的编码、古典密码、现代密码和攻击场景。
- 自动化程度高：不确定算法时可用一键解密，多层嵌套时可用递归解密。
- 结果更清晰：爆破结果会展示参数和解密路径，便于复现解题过程。
- 脚本友好：CLI 支持批量处理、文件输入输出和 JSON 输出，方便接入自己的工具链。
- 可扩展：新增普通算法只需添加算法文件，GUI 和 CLI 会自动识别。
- 辅助分析：内置 AI 分析入口，可用于整理思路、判断密文特征和辅助排查方向。
- 结果一致：GUI 和 CLI 复用同一套核心逻辑，常见解密结果和排序保持一致。
- 参数省心：算法参数会根据函数签名自动生成，不需要用户手动查复杂配置。
- 查找方便：输出区支持关键字匹配、搜索跳转和高亮，适合从大量候选结果中找 flag。
- 攻击集中：RSA、ECC、LCG 攻击入口集中在工具内，减少在多个脚本之间来回切换。
- SageMath 支持：部分高级攻击可使用本机、WSL 或在线 SageCell 执行。
- 配置持久化：字体、颜色、自动复制、算法选项卡等常用设置会保存到本地配置。
- 便于排错：异常会写入日志文件，方便定位运行问题。

## 功能概览

- 常见编码/解码：Base32、Base64、Base85、Base91、Hex、URL、Unicode、Morse、HTML 等。
- 古典密码：凯撒、维吉尼亚、栅栏、仿射、培根、Playfair、Hill、Enigma 等。
- 哈希计算：MD5、SHA1、SHA2、SHA3、BLAKE2、HMAC、CRC32 等。
- 对称密码：AES、DES、3DES、RC4、Blowfish、ChaCha20、Twofish、XOR 等。
- 非对称密码：RSA、ECC、ECDSA 签名/验签。
- 攻击辅助：RSA、ECC、LCG 常见 CTF 攻击场景。
- 自动分析：一键解密、递归解密、AI 辅助分析密文。

## 安装

需要 Python 3.8+。

```bash
pip install -r requirements.txt
```

## 启动 GUI

```bash
python main.py
```

GUI 适合手动分析：粘贴密文，选择算法，填写参数，直接查看结果。

## 使用 CLI

查看支持的算法：

```bash
python cli.py list
```

加解密示例：

```bash
python cli.py encrypt base64 "hello"
python cli.py decrypt base64 "aGVsbG8="
```

不确定密文类型时，先尝试一键解密：

```bash
python cli.py try-all "U0VGQ1RGe3Rlc3R9"
```

遇到多层编码或多层古典密码时，使用递归解密：

```bash
python cli.py recursive "U0VGQ1RGe3Rlc3R9"
```

进入攻击辅助模块：

```bash
python cli.py rsa
python cli.py ecc
python cli.py lcg
```

## 常用场景

- 不确定密文类型：先用 `try-all`。
- 多层编码嵌套：使用 `recursive`。
- 批量处理密文：使用 CLI 文件输入、批量模式或 JSON 输出。
- RSA/ECC/LCG 题目：进入对应模块，根据题目参数选择攻击方法。
- 手动调试参数：使用 GUI 更直观。

## 注意事项

- 哈希通常不可逆，只能计算摘要，不能直接“解密”。
- AES、RSA、ECC 等现代密码需要密钥、参数或满足攻击条件。
- 部分高级攻击需要 SageMath 环境。
- 本工具仅用于 CTF、密码学习、安全教育和授权分析。
